Ingegneria del software

M.T.O - Modulo di Tirocinio Online


Il progetto é stato sviluppato da un team composto di quattro persone dove ognuno aveva un ruolo. Per la realizzazione del progetto ogni settimana si tenevano dei meeting documentati. Il sistema si occupa delle richieste di tirocinio e delle relatice documentazioni. Il software è di tipo Greenfield Engineering, quindi è stato progettato partendo dall'analisi dei requistiti. Le operazioni come la richiesta del tirocinio, possono esser fatte online, questo semplifica il lavoro dei dipendenti universitari. In questo progetto io ero responsabile della revisione e consegna dei documenti. Con i miei colleghi ho realizzato il sito web, il database e la documentazione(scritta in Italiano).
Sotto ci sono i documenti:

RAD - Requirements Analysis Document

SDD - System Design Document

ODD - Object Design Document

TP - Test Plan

TCS - Test Case Specification

TIR - Test Incident Report

TSR - Test Summary Report

TER - Test Execution Report

UTP - Unit Test Plan
